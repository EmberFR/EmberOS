# 🔥 EmberOS

EmberOS is a Python-based terminal system that simulates a mini operating system with user login, admin features, utilities, and built-in games.

---

## Features

- Login & Signup system  
- Admin mode (special access)  
- Calculator  
- Dice Roller  
- Guess The Number game  
- Magic 8 Ball  
- ASCII art display  
- Account information system  

---

## About

This project was built to practice Python fundamentals including:
- Variables  
- User input  
- Loops  
- Conditional statements  
- Program structure  

---

1. Make sure you have Python installed  
2. Download or clone this repository
